# mygirl_town

------

## SetUp
```bash
## Install Python-IDE
python 2.7.14

## Install Scrapy
### upgrade pip
python -m pip install --upgrade pip
pip install Scrapy

## create the project
scrapy startproject projectname
## projectname - you can define it yourself.

## copy this code, you can run it.
scrapy crawl projectname
```
